# 🛒 E-commerce Sales Analysis Project

## 📌 Project Overview
This project analyzes an e-commerce sales dataset using Python.

The goal is to understand business performance, sales trends, and profit insights.

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

---

## 📊 Analysis Performed
✔ Total Sales & Profit calculation  
✔ Sales by Category  
✔ Profit by Category  
✔ Sales by Region  
✔ Monthly Sales Trend  
✔ Top Selling Product Categories  

---

## 📈 Key Insights
- Some categories generate high sales but low profit.
- Sales vary significantly across regions.
- Monthly sales trend shows business growth pattern.
- Certain product categories dominate the market.

---

## ▶️ How to Run
Install libraries:
pip install pandas matplotlib

Run the script:
python analysis.py

Charts will be generated and saved in the **charts/** folder.
